# Template project VUE 3

## Mock server start
```
npm install -g json-server
json-server --watch db.json
```
Server will start on  http://localhost:3000 by default

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
